Установка скрипта крайне проста.

1) Закачиваете все файлы в дирректорию sendmail на своем сайте.
2) Открываете файл config.php и выставляете необходимые настройки.
3) Скрипт готов к работе.

Этот скрипт оттестирован в течение года.
Он надежен, безопасен для сайта, хорошо упреждает поток спама.

Разработчики:
